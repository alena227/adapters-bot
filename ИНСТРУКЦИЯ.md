# Инструкция по запуску бота

## Шаг 1. Получи токен бота в Telegram

1. Открой Telegram, найди @BotFather
2. Напиши `/newbot`
3. Придумай имя и username для бота (username должен заканчиваться на `bot`)
4. Скопируй токен — он выглядит как `1234567890:AAFxxxxxxxxxxxxxxx`

## Шаг 2. Создай файл .env

В папке `adapters_bot` создай файл `.env` (скопируй `.env.example`):
```
BOT_TOKEN=вставь_сюда_свой_токен
```

## Шаг 3. Настрой Google Sheets

### 3.1. Создай Google Cloud проект

1. Зайди на https://console.cloud.google.com/
2. Нажми «Создать проект», назови его (например `adapters-bot`)
3. В боковом меню → «API и сервисы» → «Включить API»
4. Найди и включи **Google Sheets API** и **Google Drive API**

### 3.2. Создай сервисный аккаунт

1. «API и сервисы» → «Учётные данные» → «Создать учётные данные» → «Сервисный аккаунт»
2. Дай любое имя, нажми «Готово»
3. Открой созданный сервисный аккаунт → вкладка «Ключи»
4. «Добавить ключ» → «JSON» → скачается файл
5. Переименуй скачанный файл в `credentials.json` и положи его в папку `adapters_bot`
6. Скопируй email сервисного аккаунта (выглядит как `имя@проект.iam.gserviceaccount.com`)

### 3.3. Создай две Google таблицы

1. Зайди на https://docs.google.com/spreadsheets/
2. Создай таблицу **«Пробное напарничество»**
   - Поделись ею с email сервисного аккаунта (права «Редактор»)
   - Скопируй ID из URL: `docs.google.com/spreadsheets/d/ВОТ_ЭТО_ID/edit`
3. Создай таблицу **«Собрания»**
   - Поделись с тем же email
   - Скопируй ID

### 3.4. Вставь ID таблиц в config.py

Открой файл `config.py` и замени:
```python
PARTNERSHIP_SPREADSHEET_ID = "ВСТАВЬ_ID_ТАБЛИЦЫ_НАПАРНИЧЕСТВА"
MEETINGS_SPREADSHEET_ID    = "ВСТАВЬ_ID_ТАБЛИЦЫ_СОБРАНИЙ"
```
на реальные ID.

## Шаг 4. Установи Python и зависимости

1. Скачай Python 3.11+ с https://python.org и установи (галочку «Add to PATH»!)
2. Открой папку `adapters_bot` в терминале (правой кнопкой → «Открыть в терминале»)
3. Выполни команды:
```
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## Шаг 5. Запусти бота

```
python bot.py
```

Бот запустится. Напиши `/start` своему боту в Telegram — он должен ответить.

Чтобы остановить бота: `Ctrl+C` в терминале.

## Как редактировать бота в будущем

### Добавить новые собрания (напарничество)
В `config.py` в `PARTNERSHIP_MEETINGS` добавь строку:
```python
"10": "10 собрание",
```

### Открыть Летнюю интенсиву
В `config.py` измени:
```python
ACTIVE_PERIODS = ["spring", "summer"]
```
И добавь собрания в `PERIOD_MEETINGS["summer"]`.

### Добавить новый вопрос
1. В `config.py` добавь колонку в `PARTNERSHIP_HEADERS` или `MEETING_HEADERS`
2. В `bot.py` добавь новое состояние и функцию-обработчик по аналогии с существующими
3. В `sheets.py` добавь значение в нужный список `row`

## Как запускать постоянно (не только пока открыт терминал)

Для постоянной работы бота нужен сервер. Бесплатные варианты:
- **Railway** — https://railway.app (простой деплой)
- **Render** — https://render.com
- **PythonAnywhere** — https://pythonanywhere.com

Либо оставь бота работать на своём компьютере, запустив `python bot.py`.
